CSS FRAMEWORKS

Okay, so we've learned that doing layout with CSS by hand is confusing, inconsistent between different browsers, resolutions and devices and just generally a great, big pain in the button. Well...

Good news everyone! CSS Frameworks exist! A framework (or library) just means "a bunch of pre-written code", usually code that solves frequently-encountered problems common to many projects. A lazy programmer is a good programmer. Don't reinvent the wheel!

Here is a simple web page which builds off of a super-simple CSS framework called Minimal CSS. Let's take a look!

  1. Open the Minimal CSS link and take a look around the demo page.
  2. View Source ([CMD + SHIFT + ?] and type in "source" if you can't remember the keyboard shortcut) and make a list of the classes being used on the page.
  3. Open the Chrome Developer Tools and inspect various .row and .col-n elements. Where did they get those styles?
  4. From the view source page, open the link to the minimal.css stylesheet and read over the css rules defined there.

Above I've selected a div with the class col-6 from the example site. Here's what we've learned about this element:

  • It has no styles applied directly to it by an HTML style attribute
  • On line 37 of the linked stylesheet minimal.css there is a rule that applies only to the class col-6 that sets its width.
  • On line 28 of minimal.css it picks up more styles from a rule that is applied to multiple column classes.
  • On line 7 of minimal.css it gets its margin and padding from a rule applied to all elements on the page.
  • It is a div, so the browser displays it as a block element.

Exercise: Community Site Pintrest

Let's make a new web page that has a bunch of inspirational text and images for our online community members in a single place. We can use minimal.css to create a lot of small divs in multiple configurations to make it visually interesting.

  1. Set up a new HTML document
  2. Download minimal.css and link it in the head
  3. Create a styles.css and link it IMMEDIATELY AFTER your css framework. Why not before?
  4. Place the content of the page in a fixed-width, centered #container
  5. Have a full width header and footer section
  6. Create a crazy interspersed grid of captioned images and text.

Bootstrap!

Bootstrap is a more fully-featured framework that gives us pre-written CSS and JavaScript code (we'll just focus on the CSS for now). The CSS alone consists of 8000+ lines of code!!!.

This will be an exercise in ignoring complexity and zeroing to finding only the information you need to take the next step forward. (Hint: Try using [CMD + F] to search Bootstrap's CSS documentation page for each bolded term below.)

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/3.3.5/css/bootstrap.min.css">

You can make this page about anything, but if you're lacking inspiration may I suggest...

  1. Add the above link code to the head of a fresh HTML document.
  2. Browse through the documentation of the CSS portion of Boostrap.
  3. Create a simple grid system for your page.
  4. Create a section of lead body copy
  5. Use some marked text
  6. Use some text alignment classes
  7. Create a table with striped rows
  8. Use some button classes
  9. Try a few image shapes
